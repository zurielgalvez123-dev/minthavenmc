# MintHaven Store

Official store site for MintHaven MC — https://store.minthavenmc.com

Static single-file site (`index.html` + `assets/`), hosted on GitHub Pages.

Store settings live in the `CONFIG` block at the top of `index.html`; packages in `CATALOG`. Empty config values render honest "coming soon" states automatically.
